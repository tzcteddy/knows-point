## 绘制三角形有哪些方法

**方案一**
```css
.inner{
    width: 0;
    height: 0;
    border-color:transparent ;
    border-left-color: green;
    border-width: 100px;
    border-style: solid;
  }
```

**方案二**
```css
  .inner{
    width: 100px;
    height: 100px;
    background: linear-gradient(45deg,red 50%,transparent 50%);
  }
```
