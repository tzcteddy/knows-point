grid-auto-flow 属性 默认值是row，即"先行后列"。也可以将它设成column，变成"先列后行"。

设为row dense，表示"先行后列"，并且尽可能紧密填满，尽量不出现空格

设置改为column dense，表示"先列后行"，并且尽量填满空格。


      .box-grid-flow{
        grid-auto-flow: column;
      }
      .box-grid-flow-row{
        grid-auto-flow: row dense;
      }
      .box-grid-flow-column{
        grid-auto-flow: column dense;
      }
