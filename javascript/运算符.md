## 运算符

### 算术运算符
|符号|描述|
|---:|:---|
|+|加|
|-|减|
|*|乘|
|/|除|
|%|取余|
|++|递增|
|--|递减|

### 赋值操作符
|符号|描述|
|---:|:---|
|=|赋值|
|+=|加赋值(x += y) == (x = x + y)|
|-=|减赋值(x -= y) == (x = x - y)|
|*=|乘赋值(x *= y) == (x = x * y)|
|/=|除赋值(x /= y) == (x = x / y)|
|%=|取余赋值(x %= y) == (x = x % y)|

### 比较运算符
|符号|描述|
|---:|:---|
|==|相等|
|===|全等|
|!=|不等|
|>|大于|
|>=|大于等于|
|<|小于|
|<=|小于等于|

### 逻辑运算符
|符号|描述|
|---:|:---|
|&&|与|
|\|\||或|
|!|非|

### 位运算符
|符号|描述|
|---:|:---|
|&|与|
|\||或|
|~|非 (就是将原来的数值加1后取负)|
|^|异或|
|<<|左移|
|>>|右移|

+ 1、计算机里的数字是以补码的形式表示的。
+ 2、取反即：0变1，1变0。
+ 3、第一个数为符号位，0为正数，1为负数。
+ 4、正数的原码、反码、补码都一样。
+ 5、负数的反码：符号位不变，其余取反。补码：反码加1。
+ 6、0的反码补码都为0。
